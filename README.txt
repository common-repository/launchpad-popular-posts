== Description ==
This is a very simple, easy to use plugin which creates a widget that can be used to display Popular Posts, Related Posts, Featured Posts, Recent Posts with thumbnails based on the number of comments. Users can choose how many posts they want to show in the widget settings.

Features

1. Post status option.
2. Shortcode feature.
3. Display by date, comment count or random.
4. Display thumbnails, with customizable size and alignment.
5. Display excerpt, with customizable length.
6. Display from all, specific or multiple category.
7. Post type option.
7. Multiple widgets.
7. Custom CSS.

__Shortcode__

Don't want to worry about HTML tags?  You can use a shortcode in your posts, pages to dispaly Launchpad Article Feedback.

The shortcode to use is `[launchpad_popular_posts]`, `[launchpad_related_posts]`, `[launchpad_recent_posts]`.

**Examples**

`[launchpad_popular_posts]`
`[launchpad_related_posts]`
`[launchpad_recent_posts]`
== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or find `Launchpad Popular Posts` and install the plugin through the WordPress plugins screen directly.

2. Activate the plugin through the 'Plugins' menu in WordPress.

3. Use one of the two usage methods (HTML, Shortcode) within the content of your posts or pages.   

4. Use either the HTML or Shortcode methods inside your text widgets.
